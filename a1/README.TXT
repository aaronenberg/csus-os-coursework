INSTRUCTIONS ON HOW TO RUN THE OBSERVER PROGRAM:
    
    COMMAND TO RUN OBSERVER IN STANDARD REPORT MODE (PART B):
        ./observer

    COMMAND TO RUN OBSERVER IN SHORT REPORT MODE (PART C):
        ./observer -s

        
    COMMAND TO RUN OBSERVER IN LONG REPORT MODE (PART D):
        ./observer -l interval duration
            interval (integer) - specifies how often the load average should be sampled
            duration (integer) - specifies the total time over which the load average is sampled
